# schedule_management_app_lsj_spring9_ch3
일정 관리 앱
* Author : LEE SEOJUN

## 프로젝트 소개
일정 관리 API를 구현
사용자는 일정을 생성, 조회, 수정, 삭제를 할 수 있습니다.
사용자는 일정에 댓글을 생성할 수 있습니다.
